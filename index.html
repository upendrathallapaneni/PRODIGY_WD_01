About

Why this page works

This landing page is built to meet the internship brief exactly — a fixed nav that changes on scroll, hover states on every link, and a layout that stays usable at any screen size.

🎯

Fixed navigation

The navbar stays at the top of the screen as you scroll through all sections — always accessible, never in the way.

✨

Scroll-aware style

Background, text color, and height all transition smoothly when the page scrolls past the hero — no jarring jumps.

📱

Fully responsive

A hamburger menu appears on small screens, the layout reflows into single columns, and nothing overflows.

Services

What's built in

Every feature required by the brief is implemented — no libraries, no frameworks, just clean vanilla HTML, CSS and JavaScript.

01

Scroll detection

A single scroll event listener watches window.scrollY and toggles a .scrolled class on the nav element.

JavaScript
02

Hover effects

CSS transitions handle all link highlights. The active section's nav link stays highlighted using an Intersection Observer.

CSS
03

Smooth scroll

All anchor links scroll smoothly to their target section. The nav height offset is accounted for with scroll-margin-top.

HTML + CSS
04

Mobile menu

The desktop links collapse into a hamburger on narrow screens. Tapping it reveals a full-width dropdown with animated icon.
