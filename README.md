# PRODIGY_WD_01: Responsive Landing Page

A modern, fluid landing page website built to showcase responsive design principles. The layout adapts seamlessly to any viewport, providing an optimized user experience across mobile devices, tablets, and large desktop screens. Developed as part of my Web Development Internship at Prodigy InfoTech.

## 🚀 Features
- **Fluid Grid Layouts:** Built entirely using modern CSS Flexbox and Grid layout systems.
- **Media Queries:** Tailored responsive breakpoints ensuring zero layout breakage on small viewports.
- **Clean Structure:** Uses highly semantic HTML5 elements for optimal accessibility.
- **Hover Transitions:** Smooth visual interactive feedback effects on navigation links and action elements.

## 🛠️ Technologies Used
- **HTML5:** Structural layout markup.
- **CSS3:** Custom styles, media queries, flexbox, and smooth transition properties.

## 📦 How to Run the Project
1. Clone this repository to your machine.
2. Double-click the `index.html` file to launch it instantly in any modern web browser.
